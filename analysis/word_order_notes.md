# Word Order Notes

## 仮説

字義的項から慣用的比喩項へ進む順序は、Shen 1998の典型項→非典型項という傾向と接続できる可能性がある。

ただし、日本語で同じ傾向が成り立つかは未検証である。

## 予備分析

| 順序 | 自然さ | くびき語法らしさ | dead metaphor再前景化 |
|---|---|---|---|
| 荷物と不安を抱える | 高 | 高 | 高 |
| 不安と荷物を抱える | 中 | 中 | 中。先に比喩項が来るため物理基盤の前景化は弱くなる可能性 |
| 涙と本音をこぼす | 高 | 高 | 高 |
| 本音と涙をこぼす | 中 | 中 | 中。慣用表現から字義的項へ戻る順序になる |
| 部屋と気持ちを整理する | 高 | 高 | 中〜高 |
| 気持ちと部屋を整理する | 中 | 中 | 中 |

## 暫定結論

字義的項→慣用的比喩項の順序は、くびき語法らしさと再前景化を観察しやすくする可能性がある。

ただし、これは現段階では作例に基づく予備判断であり、実例・自然性判断・語順入れ替えテストが必要。

## 危険な点

- 日本語は語順の自由度が英語と異なる。
- 助詞や文脈によって自然さが変わる。
- Shen 1998の結果を日本語へ直接一般化しない。
