# Dead Metaphor Reactivation Framework

## 目的

このファイルは、「くびき語法による dead metaphor の再活性化」という仮説を、発表で扱える分析枠組みに落とし込むための作業メモである。

ここでいう「再活性化」は、読者の心理処理を直接断定する語ではなく、意味記述上の「再前景化」「再具体化」「比喩性の可視化」として扱う。

## dead metaphor 再活性化とは何か

本研究では、dead metaphor を暫定的に「慣用化・通常化により、通常は字義的・身体的基盤が前景化されにくくなった比喩表現」とする。

dead metaphor 再活性化とは、そうした慣用化した比喩表現が、別の字義的用法と並置されることで、もとの身体的・空間的・物理的基盤を見えやすくする現象である。

例:

- `荷物を抱える` と `不安を抱える`
- `涙をこぼす` と `本音をこぼす`
- `部屋を整理する` と `気持ちを整理する`

ただし、現時点では「dead metaphorが本当に心理的に再活性化される」とは言わない。観察できるのは、字義的項と慣用的比喩項が同じ共有語句に支配される構造であり、その構造が比喩的基盤を前景化しうるという点である。

## くびき語法がなぜ再活性化に関わる可能性があるのか

Shen 1998では、くびき語法において共有述語が複数の項を一時的なアドホックカテゴリーへ統合するとされる。

本研究では、この考えを次のように拡張して検討する。

1. 共有語句は、字義的項と慣用的比喩項を同じ統語的位置に置く。
2. 字義的項は、共有語句の基本義・身体的基盤を明示しやすい。
3. 慣用的比喩項は、通常は慣用表現として処理される。
4. 両者が同じ共有語句に支配されることで、慣用的比喩項の比喩性が可視化される可能性がある。

このため、くびき語法は単なる省略や言葉遊びではなく、慣用化した比喩表現を再び物理的・身体的意味の側から見せる装置として働く可能性がある。

## 字義的項と慣用的比喩項の関係

| 項目 | 説明 | 例 |
|---|---|---|
| 字義的項 | 共有語句の基本義に典型的に合う項 | 荷物、涙、部屋、ドア |
| 慣用的比喩項 | 共有語句と慣用化した比喩関係を作る項 | 不安、本音、気持ち、心 |
| 共有語句 | 両項に同時に関係する語句 | 抱える、こぼす、整理する、開く |
| 再前景化 | 慣用的比喩項の字義的・身体的基盤が見えやすくなること | 不安が「抱えられるもの」として具体化される |

重要なのは、共有語句が一方的に項をカテゴリー化するだけでなく、非典型項が共有語句の意味範囲を押し広げる点である。ただし、この「押し広げ」は心理処理ではなく、意味記述上の関係として扱う。

## 非典型項の分類

| 型 | 非典型項の特徴 | 例 | 中心性 |
|---|---|---|---|
| dead metaphor 再活性化型 | 慣用化した比喩表現の一部 | 不安を抱える、責任を背負う | 最重要 |
| metaphor型 | その場で比喩的に処理される | swallowed kisses | 補助 |
| metonymy混合型 | 身体部位語などが内面を指す | heart in broke his heart | 補助 |
| idiom literalization型 | 慣用句の字義的側面が見えやすい | lost his temper | 重要 |
| polysemy型 | 共有語句の別語義が切り替わる | fired his gun and his assistant | 対照例 |
| frame shift型 | 物理フレームと社会・心理フレームが切り替わる | covered the table and the scandal | 補助 |
| 距離近すぎ型 | 両項が近すぎて通常の並列になる | 荷物と鞄を抱える | 除外寄り |
| 距離遠すぎ型 | 両項が遠すぎて統合が困難 | 荷物と民主主義を抱える | 除外寄り |

## 語順の役割

字義的項から慣用的比喩項へ進む順序は、Shen 1998の「典型項が非典型項より先に置かれやすい」という観点と接続できる。

予備仮説:

- 字義的項 -> 慣用的比喩項: 共有語句の基本義が先に立ち、慣用的比喩項の再前景化が観察しやすい。
- 慣用的比喩項 -> 字義的項: 慣用表現が先に処理されるため、字義的基盤の可視化が弱まる、または効果が後から生じる可能性がある。

例:

| 順序 | 暫定評価 |
|---|---|
| 荷物と不安を抱える | 自然。再前景化が強い |
| 不安と荷物を抱える | 可能だが、効果はやや弱い |
| 涙と本音をこぼす | 自然。再具体化が強い |
| 本音と涙をこぼす | やや文体的。語順効果の検討対象 |

ただし、日本語は英語より語順の自由度が高いため、Shen 1998の結果を直接一般化しない。

## 意味的距離の役割

意味的距離は、くびき語法らしさと再前景化の強さを左右する可能性がある。

| 距離 | 特徴 | 例 | 暫定評価 |
|---|---|---|---|
| close | 両項が同じ物理領域にある | 荷物と鞄を抱える | 通常の並列。くびき語法らしさは低い |
| moderate | 物理対象と抽象対象が適度に離れる | 荷物と不安を抱える | くびき語法らしさと再前景化が出やすい |
| too_far | 共有語句で統合しにくい | 荷物と民主主義を抱える | 不自然。文脈なしでは分析しにくい |
| unclear | 文脈依存 | 火と怒りを抑える | 実例確認が必要 |

## 「読み直し」を断定しないための表現

避ける表現:

- 読者は必ず読み直す。
- 字義的意味が必ず先に処理される。
- dead metaphorが心理的に復活する。

使う表現:

- 字義的基盤が再前景化される可能性がある。
- 身体的基盤が再具体化される。
- 慣用的比喩項の比喩性が可視化される。
- 共有語句の基本義と慣用的比喩義が同一構文内で並置される。
- 意味記述上、物理的フレームと抽象的フレームが同時に関係づけられる。

## 発表で安全に言える主張

- くびき語法では、共有語句が複数の項に同時に関係する。
- Shen 1998は、この構造をアドホックカテゴリーと典型性の観点から説明している。
- 字義的項と慣用的比喩項が並ぶ例では、慣用的比喩項の字義的・身体的基盤が再前景化される可能性がある。
- その効果は、非典型項の性質、語順、意味的距離によって変わる可能性がある。
- Goossens 1990のmetaphtonymyは、metaphorとmetonymyが重なる一部の例に限って補助的に使える。

## 危険な主張

- 危険: くびき語法は常に dead metaphor を再活性化する。
- 危険: くびき語法をmetaphtonymyと同一視する。
- 危険: すべての慣用的比喩項が dead metaphor である。
- 危険: 作例だけで日本語くびき語法の一般的性質を主張する。
- 危険: 心理実験なしに、読者の処理順序を断定する。
- 危険: frame shift型やpolysemy型まで、すべてdead metaphor再活性化型に入れる。
