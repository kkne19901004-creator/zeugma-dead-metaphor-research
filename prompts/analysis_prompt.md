# Analysis Prompt

以下の例文を、くびき語法による dead metaphor 再前景化の観点から分析してください。

分析項目:

1. 共有語句
2. 字義的項
3. 慣用的比喩項
4. 共有語句の基本義
5. 慣用的比喩項における共有語句の意味
6. dead metaphor再前景化の可能性
7. Shen 1998のアドホックカテゴリーとの関係
8. Goossens 1990のmetaphtonymyを補助的に使う必要があるか
9. 分類
10. 危険な点

分類候補:

- dead metaphor再活性化型
- metaphor型
- metonymy混合型
- idiom型
- polysemy型
- frame shift型
- 抽象化型

注意:

- 読者の心理処理を断定しない。
- 「読み直し」ではなく「再前景化」「再具体化」「比喩性の可視化」を使う。
- metaphtonymyは中心概念にしない。
