# Slide Generation Prompt

以下の研究テーマについて、5枚程度の発表資料案を作ってください。

研究テーマ:
くびき語法による dead metaphor の再前景化

中心仮説:
くびき語法では、共有語句が字義的項と慣用化した比喩的項に同時に関係することで、慣用化した比喩表現の字義的・身体的基盤が再前景化される場合がある。

構成:

1. 問題設定
2. 先行研究
3. 中心仮説
4. 分析観点
5. 予備分析と今後の課題

各スライドに含めるもの:

- スライドタイトル
- 箇条書き本文
- 発表者ノート

注意:

- Shen 1998を主軸にする。
- Goossens 1990は補助的に扱う。
- くびき語法をmetaphtonymyと同一視しない。
- 危険な主張と安全な主張を分ける。
- 日本語例は作例の可能性があるため、実例確認が必要と書く。
