# Example Collection Prompt

以下の条件で、英語・日本語の例文候補を集めてください。

研究テーマ:
くびき語法による dead metaphor の再前景化

収集したい例:

- 字義的項と慣用的比喩項が同じ共有語句に支配される例
- 共有語句が物理的意味と抽象的意味を同時に持つ例
- dead metaphorまたはconventional metaphorの字義的基盤が見えやすくなる例

記録する列:

- id
- example
- shared_expression
- typical_item
- atypical_item
- literal_side
- figurative_side
- classification
- source
- notes

注意:

- 作例と実例を分ける。
- 日本語例は必ず実例確認が必要。
- 自然さが低い例は「低」と明記する。
- くびき語法か単なる多義語かを区別する。
