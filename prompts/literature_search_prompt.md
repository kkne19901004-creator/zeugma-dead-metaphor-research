# Literature Search Prompt

以下の研究テーマについて、関連文献を探してください。

研究テーマ:
くびき語法による dead metaphor の再活性化

中心仮説:
くびき語法では、共有語句が字義的項と慣用化した比喩的項に同時に関係することで、通常は意識されにくい比喩表現の字義的・身体的基盤が再前景化される場合がある。

探してほしい領域:

- dead metaphor
- conventional metaphor
- metaphor revitalization
- metaphor reactivation
- idiom literalization
- idiom variation
- defamiliarization
- foregrounding
- zeugma
- syllepsis
- ad hoc category
- metaphorical categorization
- semantic distance
- frame semantics

出力形式:

1. 文献情報
2. 要旨
3. この研究に使える点
4. 危険な点
5. 次に読むべき文献

注意:

- くびき語法をmetaphtonymyと同一視しない。
- 文献に書かれていないことは断定しない。
- dead metaphor再活性化の根拠になる文献を優先する。
