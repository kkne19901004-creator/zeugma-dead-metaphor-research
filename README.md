# Zeugma and Dead Metaphor Reactivation Research

## 目的

このリポジトリは、研究テーマ「くびき語法による dead metaphor の再活性化」を管理するための作業場です。

中心は、くびき語法を単なる言葉遊びとしてではなく、共有語句が字義的項と慣用化した比喩的項に同時に関係することで、通常は見えにくい比喩の字義的・身体的基盤を再前景化する可能性を検討することです。

## 研究テーマ

**くびき語法による dead metaphor の再活性化**

## 中心仮説

くびき語法では、共有語句が字義的項と慣用化した比喩的項に同時に関係することで、通常は意識されにくい比喩表現の字義的・身体的基盤が再び前景化される場合がある。つまり、くびき語法は dead metaphor を再活性化する装置として働く可能性がある。

この仮説は、現段階では確定主張ではなく、予備分析のための作業仮説である。

## 重要な立場

- くびき語法をmetaphtonymyと同一視しない。
- Shen 1998を主軸にし、Goossens 1990は補助的に扱う。
- 読者の心理処理を断定しない。
- 「読み直し」よりも「再前景化」「再具体化」「比喩性の可視化」という意味記述を優先する。
- 日本語例は作例を含む可能性があるため、今後コーパスや実例で確認する。

## 現時点の成功条件

- 危険な主張と安全な主張を区別できる。
