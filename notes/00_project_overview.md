# Project Overview

## 研究テーマ

くびき語法による dead metaphor の再活性化

## 発想

くびき語法では、共有語句が複数の項に同時に関係する。その項の一方が字義的対象で、もう一方が慣用化した比喩的対象である場合、慣用化して見えにくくなった比喩の字義的・身体的基盤が再前景化される可能性がある。

## 例

- 荷物と不安を抱える
- 涙と本音をこぼす
- 部屋と気持ちを整理する

## 分析の基本単位

| 項目 | 説明 |
|---|---|
| 共有語句 | 複数の項に同時に関係する語句 |
| 字義的項 | 共有語句の基本義に典型的に合う項 |
| 慣用的比喩項 | 慣用化した比喩として共有語句と結びつく項 |
| 再前景化 | 慣用的比喩の字義的・身体的基盤が見えやすくなること |
