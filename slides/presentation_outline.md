# Presentation Outline

## タイトル案

くびき語法による dead metaphor の再前景化

## 発表の目的

Shen 1998のアドホックカテゴリーの枠組みを基盤に、くびき語法が慣用化した比喩表現の字義的・身体的基盤を再前景化する可能性を検討する。

## 構成

1. 問題設定
2. 先行研究
3. 中心仮説
4. 分析観点
5. 予備分析
6. 危険な主張
7. 今後の課題

## 発表で強く言えること

- くびき語法では共有語句が複数項に関係する。
- Shen 1998は、共有述語がアドホックカテゴリーを作ると論じる。
- dead metaphor再前景化は現時点では発展仮説である。

## 発表で避けること

- くびき語法をmetaphtonymyと同一視すること
- 読者が必ず読み直す
- 日本語例から一般化
- dead metaphorの定義を未確認のまま固定
